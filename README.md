## üöÄ Getting Started

### Prerequisites
* Python 3.8+
* pip

### Installation

1. **Clone the repository:**
```bash
   git clone https://github.com/<your-username>/utca-conference-website.git
   cd utca-conference-website
```

2. **Install dependencies:**
```bash
   pip install -r requirements.txt
```

### Run Locally

1. **Start the server:**
```bash
   python app.py
```

2. **Open your browser:** Navigate to `http://127.0.0.1:5000`
